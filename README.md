# GEV and MEVD Modeling of Extreme Precipitation with Statistical Downscaling Applied to MEVD

This repository contains Python codes for modelling extreme precipitation using the 
Generalized Extreme Value (GEV) distribution and the Metastatistical Extreme Value Distribution (MEVD) [1,2]. 

The repository also includes tools for statistical downscaling of precipitation data and 
error modelling of satellite-based precipitation products.

## References

[1] Marani M, Ignaccolo M (2015). *A Metastatistical Approach to Rainfall Extremes*. Advances in Water Resources, 79: 121–126. https://doi.org/10.1016/j.advwatres.2015.03.001

[2] Marra F, Borga M, Morin E (2020). *A Uniﬁed Framework for Extreme Subdaily Precipitation Frequency Analyses Based on Ordinary Events*. Geophysical Research Letters 47: 1-8. https://doi.org/10.1029/2020GL090209
